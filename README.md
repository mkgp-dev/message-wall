# message-wall
A simple Mini Message Board project submitted for The Odin Project course.

### Built With

- **Node.js** ~ Runtime environment for the backend.
- **Express** ~ Web framework handling routing and middleware.
- **TypeScript** ~ Strongly-typed JavaScript for safer server code.
- **EJS** ~ Templating engine for server-rendered views.
- **Tailwind CSS** ~ Utility-first styling, compiled via `@tailwindcss/cli`.
- **DaisyUI** ~ Tailwind component library for faster UI building.
- **nanoid** ~ Tiny, secure ID generator for messages.
